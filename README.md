# NF-eBOT REST API
RestAPI builded with NodeJS/Restify used in NF-eBOT project.

## # Documentation ##
After run API, enter in ```http://localhost:8080```.

-------

## # For start and stop ##
run ``` grunt serve ``` or ``` grunt stop ```

-------

## # Before start api development job ##
run ``` grunt dev ```

-------

## # For update documentation ##
run ``` grunt doc ```
